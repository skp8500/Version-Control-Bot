─── WHAT THIS APP IS ────────────────────────────────────────

A mini GitHub where users can:
- Browse repos and files without logging in
- See a visual commit graph (like a node graph with arrows)

- Upload their own code as a repo (zip or folder)
- Edit files directly in the browser
- Push changes and detect conflicts

Never show a login page on load. Never block reading.

  → Edit a file
  → Upload files
  → Resolve a conflict

Implement this with a useAuth() hook and requireAuth(action) pattern.
