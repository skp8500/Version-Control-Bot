## FEATURE SPECS

── AUTH ──────────────────────────────────────────────────
Login / Signup page with email + password.
On success → JWT returned → stored in localStorage.
All API calls send Authorization: Bearer <token> header.
Protected routes redirect to /login if no token.

── DASHBOARD ─────────────────────────────────────────────
Shows all repos for logged-in user as cards.
Each card: repo name, description, last commit message, commit count.
"+ Create Repo" button → modal → name + description → POST /repos.
Clicking a repo card → goes to /repos/:id view.

── REPO VIEW ─────────────────────────────────────────────
Left panel: file tree. Right panel: file content viewer.
Clicking a file → opens inline editor.
User edits → "Stage Changes" → diff shown.
"Push Commit" → commit message modal → POST /repos/:id/commit.
On push backend: diffs new content vs HEAD line by line.
Clean → new commit. Conflict → 409 + conflict details.

── CONFLICT RESOLUTION ───────────────────────────────────
Split-pane: LEFT = yours (green highlights), RIGHT = incoming (red).
Options: [Keep Mine] [Keep Theirs] [Edit Manually]
After resolve → conflict marked resolved → new commit created.
AI bot explains why conflict happened.

── GRAPH VISUALIZATION TAB ───────────────────────────────
D3 force-directed graph of full commit history.
NODES: Green=HEAD, Blue=normal, Red=conflict resolved, Gray=old.
EDGES: Arrows child→parent. Main=solid. Merged=dashed.
Click node → popup: hash, author, timestamp, files changed, diff (green/red lines).
Zoom/pan with d3.zoom. Toggle grid. Animate path. Steps mode.
Toolbar: MODE [Drag][+Node][Edge][Delete] ALGO [Dijkstra][BFS]
         OPTIONS [Toggle Grid][Animate][Steps][Reset] ZOOM [-][X%][+]

── AI BOT PANEL ──────────────────────────────────────────
Floating bottom-right, collapsible.
Fires after every action. Explains internals in 4-6 sentences.
Backend route: POST /api/explain → calls Claude API.
Secret: ANTHROPIC_API_KEY in Replit Secrets.
